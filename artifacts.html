Structures: Resource Profiles

These define constraints on FHIR resources that need to be complied with by conformant implementations

PACIO Bundled Cognitive Status An point in time bundle of cognitive status observations for a patient.
PACIO Cognition Questionnaire Response A completed, or sections of a completed, CMS PAC assessment.
PACIO Cognitive Status An exchange of cognitive status data for a patient.
Use of Device A device used by a Patient during a cognitive status assessment.
